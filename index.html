Vibe Coding Showcase

  • Weather Card

    An animated weather card that displays different weather conditions. The card should visually represent the following weather conditions with distinct animations: Wind, Rain, Sun, and Snow. The card has a dark background and the JavaScript includes a way to switch between the different weather conditions to demonstrate the animations for each.

  • Todo List

    A simple and clean todo list application. Users can add new tasks, mark them as complete, and delete them. The interface is designed to be intuitive and responsive, adapting to different screen sizes.
